#########Running the code#################

Steps:

1. Open the html file in chrome browser.

NOTE: The floor lighting effect is generated randomly for each tile. Please refresh if its not visible on first try.

#############Compatibility################

This code was tested on Chrome browser, and works fine on it. (Do not use Firefox, things may go weird on it).


#############Controlz####################

The camera controls are same as the assignment demands

1: Moves camera up
2: Moves camera down
3: Moves camera left
4: Moves camera right
5: Zoom in
6: Zoom out

Additional controls to interact with the object

w: Moves object left
s: Moves object right
a: Moves object back
d: Moves object forward

7: Rotates arms
8: Rotates pedals
9: Rotates wheels



#############Bonus Tasks###################

Click on 'Map' Button: This brings another view of the scene (From the top, with everything zoomed out). The object is surrounded by a red colored square window.
Click on 'Populate' buttom to add cubes of varying sizes and orientations to the scene
Click on 'Do Not Press!!' button.
